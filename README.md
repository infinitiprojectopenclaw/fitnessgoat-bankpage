# FitnessGoat Bankpage

Elite fitness equipment e-commerce bankpage.

## Features
- 8 products with HTTPS images
- 41 countries supported
- Responsive design
- 0 console errors
- IIFE JavaScript encapsulation
- pushState/popstate navigation
- 2.8s place order animation

## Deploy to Netlify

1. Push to GitHub
2. Connect to Netlify
3. Deploy from main branch

## Tech Stack
- Pure HTML/CSS/JS (no framework)
- Single-file architecture (index.html)
- Image URLs: DALL-E HTTPS
